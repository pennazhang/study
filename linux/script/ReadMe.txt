unzip

使用方法：

- 将脚本unzip.sh拷贝到某个主目录，运行下面的脚本，可以将该组目录下所有的*.zip文件找出来，并在其所在的目录下进行解压。

 - For example：
	cd /git/umixers
	cp /git/study/linux/script/unzip/unzip.sh . 
	find . -name "*.zip" | xargs ./unzip.sh
